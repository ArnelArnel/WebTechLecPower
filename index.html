Hypertext Transfer Protocol

Hypertext Transfer Protocol

  • It is a stateless communication protocol
  • It also do not keep information about clients in between requests
  • HTTP resources are identified using URI's (RFC 3986) or specifically HTTP URL's
  • Scheme (http or https)
  • Authority - user information/authentication credentials,host
  • Query - typically provides as key value (=) pairs with ampersand (&) separated between key value pairs.
