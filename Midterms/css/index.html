What is CSS?

  • CSS stands for Cascading Style Sheet which is use to style an HTML Document.
  • CSS is a language that describes the presentation of Web pages, including colors, layout and fonts which typically see in a Web pages of a webpage.
  • CSS files can be stored External Stylesheets, Internal Stylesheets and Inline Stylesheets.

History

  • The first draft of Cascading HTML Style Sheets was published by Hakon Wium Lie on October 10, 1994.
  • Bert Bos was the first who respond the first draft of CSS which that time he was building the Argo. Argo is a highly customizable browser with style sheets which he decided to join forces with Hakon Wium Lie.
  • November 1994 CSS proposal was presented at the Web Conference. The following are the topics being discuss according to WC3.
    1. The concept of a balance between author and user preferences was novel. A fictitious screen shot showed a slider with the label user on one side and author on the other. By adjusting the slider, the user could change the mix of his own preferences and those of the author.
    2. CSS was perceived by some as being too simple for the task it was designed for. They argued that to style documents, the power of a full programming language was needed. CSS went in the exact opposite direction by making a point out of being a simple, declarative format.

Version History

  • CSS 1
    • CSS level 1 specification was first published on December 17, 1996. Hakon Wium Lie and Ber Bos credited as original developers.
    • A simple style sheet mechanism
    • Allows authors and readers to attach styles to HTML documents.
    • Human readable and writable, and expresses style in common desktop publishing terminology.
  • CSS 2
    • CSS level 2 specification was first published as a recommendation in May 1998 and developed by the W3C.
    • Includes a number of new capabilities like absolute, relative, and fixed positioning of elements and z-index, the concept of media types.
    • Support for aural style sheets that which later replaced by the CSS 3 speech modules and bidirectional text, and new font properties such as shadows.
  • CSS 2 Revision 1
    • CSS level 2.1 is a style sheets language that allows authors and users to attach stle to structured documents.
    • CSS 2.1 first became a Candidate Recommendation on February 25, 2004.
    • CSS 2.1 fixed errors in CSS 2
  • CSS 3
    • CSS level 3 started in 1998, and stil under development until 2014.
    • CSS3 aims at extending CSS 2.1.
    • CSS3 has more support for gradients with the image tag.
      • Allows you to display smooth transitions between multiple colors using browser instead of image (lessen downloading time).

    CSS Selector

    • Selector
      • Patterns used to select the element(s).
    • Selector Syntax
      • Selector Group
    • Simple Selectors
        • A smaller part of a selector in CSS3.
        • universal selectors
          • Written as "*" that matches the name of any element type.
          • *[lang=fr]
        • type selectors
          • Matches the element type name of a document language.
          • h1 {font-famly: sans-serif}
        • ID selectors
          • Based on identifier, element instance matching.
          • #nav{}
        • class selectors
          • Matches subset of "class" value, there must be dot (.) before each value.
          • .past{color: black}
        • attribute selectors
          • Author's specified rules that match elements having attributes defined in the source documents. Can match in four ways:
          • [att]
          • [att=val]
          • [att~val]
          • [att|=val]
    • Psuedo Classes
      • Used to define a special state of an element.
        • dynamic psuedo-classes
        • target psuedo-classes
        • language psuedo-classes
        • UI element states psuedo-classes
        • structural psuedo-classes
        • negotiation psuedo-classes
